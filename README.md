#####   设置GOLANG环境变量

* GO111MODULE 最好设置到环境变量中


Powershell下:

* Enable the go modules feature


    $env:GO111MODULE="on"
       
* Set the GOPROXY environment variable


    $env:GOPROXY="https://goproxy.io"
